# Tic Tac Toe AI (Minimax)

## 🎮 Overview
An AI-powered Tic Tac Toe game developed using Python and Tkinter. The game allows a human player to compete against an intelligent AI opponent using the Minimax algorithm.

---

## 🧠 AI Concept
The project implements the **Minimax algorithm**, a classic Artificial Intelligence strategy used in two-player games.  
The AI evaluates all possible future moves and selects the optimal decision to either maximize its winning chances or minimize the player's advantage.

---

## 🎯 Features
- Player vs AI gameplay
- Three difficulty levels:
  - Easy: Random moves
  - Medium: Mixed strategy (Random + Minimax)
  - Hard: Full Minimax algorithm (Unbeatable AI)
- Interactive GUI using Tkinter
- Real-time win/draw detection
- Score tracking system
- Restart game functionality
- AI move delay for realism

---

## ⚙️ Technologies Used
- Python
- Tkinter (GUI)
- Minimax Algorithm (AI)

---

## 🎮 Game Logic
- The board state is evaluated after every move
- AI recursively explores all possible game states
- Scores are assigned:
  - Win: +1
  - Loss: -1
  - Draw: 0
- The best move is selected based on optimal score
